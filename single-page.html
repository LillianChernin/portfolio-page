My Portfolio

Play Strings

March 2018

Play Strings is a game that teaches a player with no musical background how to play a song on the violin, viola or cello. It has a simple scrolling interface to guide the player on what note to play and when. It uses Web Audio API to implement pitch detection in order to give the player feedback on whether they are playing the right notes. When the player plays the correct note at the right time their score increases.

  • Node.js
  • Express
  • MongoDB
  • Javascript
  • Web Audio API
  • CSS
  • jQuery
  • EJS
  • Heroku

67k Teddy Bears

February 2018

67k Teddy Bears is a donation management application to maximize effectiveness of donations in the wake of calamities and increase transparency. Non-profit organization admins can create Relief Efforts that detail a specific list of items that are needed in the wake of a calamity (like a house fire or flooding). Donors can then go to these Relief Efforts and fulfill specific needs (like size 8 men's shoes, etc.) as they can and see in real time the impact of their donations on the overall Relief Effort. The goal is to encourage donors to donate and make sure that the right resources go to those who need them.

Technologies Used

  • Django
  • Python
  • PostresSQL
  • Javascript
  • Bootstrap
  • CSS
  • Heroku

Lily's Library

February 2018

Lily's Library is a personal library application where all the books from my oversized library can be cataloged. Users can search for books, see what shelf a book is located on, place books on hold, borrow books, add books to library and return books. Lily's Library is a decoupled application with the back-end being deployed on Heroku and serving as an API for the React.js front-end that was deployed via Surge.
